# ssh_cracker
sshのパスワード解析ツールです。辞書攻撃、総当たり攻撃、どちらも可能です。<br>
※使用する際は自己責任でお願いします＾＾<br>

## 動作可能OS
* Linux
* Windows 10 ~
* Other Unix...

## インスール方法
```
sudo apt install libssl-dev libssh-dev
git clone https://github.com/ware255/ssh_cracker.git
cd ssh_cracker
make
```
